## Installation

### RubyGems

```
$ gem install fluent-plugin-latlng
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-latlng"
```

And then execute:

```
$ bundle
```

You can generate configuration template:

```
$ fluent-plugin-config-format filter latlng
```
